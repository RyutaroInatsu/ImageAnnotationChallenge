```shell

```

## From browser
Access to the link: http://127.0.0.1:5000/

## Use from API
Run this script.
```shell
python send_api.py
```

### Remarks
- You can't upload greater than 10MB image.

### Response
- Status code: 200 is "Success"
- Status code: 400 is "Bad Request"
- Status code: 413 is "Max Content Length Over"
